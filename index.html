Hello there.

This plugin simply animates a property (color, backgroundColor, width - whatever) to a set value, and back again.

I originally wrote it to do some highlighting of text after a user interaction, but then realised it would be much better to be able to animate any property.

It /DOES NOT/ use jQuery UI - I didn't want to have to load a load of code i wouldn't use, and this is much more flexible that the "highlight" function of jQuery UI. Using ss.animateProperty

It's easy to use in the same fashion you normally use a plugin, it returns the original object(s) so is nice and chainable.

  
    var el = $('#yourElement');
    el.animateProperty({color:'red'});

or just

  $('#yourElement').animateProperty({color:'red'});
 
if you fancy.
